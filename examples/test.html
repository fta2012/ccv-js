Demos

Click the demo button in each section to run.

Tips:

  • Requires a webcam on desktop chrome or firefox with a decent CPU.
  • If you want to try your own images you can change source to an imgur link or "WEBCAM". Animated imgur gifs are loaded as mp4.
  • These demos aren't using webworkers so the slower ones will freeze your browser.
  • Most examples should run in under 5 secs (except for DPM).
